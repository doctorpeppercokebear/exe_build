## 25.0.6 (2025.10.27)
**Update 사항**

### 주요 기능 개선
1. **ESP32-S3 포트 자동 감지 및 선택**
   - USB 연결 시 ESP32-S3 장치를 자동으로 인식
   - CP210x, CH340 등 일반적인 USB-UART 드라이버 지원
   - 포트 선택 없이 바로 펌웨어 업로드 가능

2. **5초마다 자동 포트 재검색**
   - USB 연결/해제 실시간 감지
   - 새로운 장치 연결 시 자동 알림
   - 업로드 중에는 포트 검색 일시 중지

3. **업로드 버튼 클릭 시 자동 재검색**
   - 포트가 없으면 자동으로 재검색 시도
   - 명확한 오류 메시지 제공 (USB 케이블 확인 안내)

### 버그 수정
4. **ESP32-S3 부트로더 주소 수정**
   - 0x1000 → 0x0으로 변경
   - 펌웨어 업로드 후 기기가 정상 작동하지 않던 문제 해결

5. **Stub flasher JSON 파일 오류 해결**
   - `--no-stub` 옵션 추가
   - PyInstaller 빌드 시 JSON 파일 누락 문제 우회

6. **사용자 경험 개선**
   - 프로그램 시작 시 장치 준비 상태 자동 안내
   - 포트 감지 실패 시 친절한 안내 메시지
   - 로그 메시지 명확화

### 기술적 변경사항
- esptool 명령어에 `--no-stub` 옵션 추가
- Bootloader 플래시 주소: 0x0 (ESP32-S3 표준)
- Partitions 주소: 0x8000 (변경 없음)
- Firmware 주소: 0x10000 (변경 없음)
